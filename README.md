# Kgaje Auto | Premium Mechanical Specialist

A high-end, responsive single-page web platform designed for **Kgaje Auto**. This platform showcases premium automotive workshop engineering, precision diagnostics, mechanical restorations, and provides an interactive digital concierge booking system for clients.

## 🛠️ Features

* **Dynamic Hero Slideshow:** A fully automated, fade-transition background slideshow showcasing high-caliber workshop visuals.
* **Fully Responsive Design:** Tailored styling for seamless cross-platform performance across desktops, tablets, and smartphones.
* **Fluid Mobile Navigation:** Includes an interactive, amber-accented sliding hamburger navigation drawer for touch-screen devices.
* **Concierge Booking System:** An asynchronous contact form with an immediate, personalized success window that prevents raw page reloads.

## 🧰 Tech Stack

* **Frontend:** HTML5, CSS3 (Custom Media Queries, Flexbox, Grid)
* **Interactivity:** Vanilla JavaScript (ES6+)
* **Icons:** FontAwesome

## 🚀 Deployment

The production build of this platform is continuously deployed via Netlify and can be accessed live. https://kgaje-auto.netlify.app/
